## <i>Resolver um problema usando Backtracking</i>
<hr>
<br>
</div>

#### Dado um conjunto <i>S</i> de elementos únicos, calcule todos os seus subconjuntos de <i>N</i> elementos.

Exemplos:

| Entradas               | Saída                  |
|------------------------|------------------------|
| S = [1, 2, 3], N = 2   | [1, 2], [1, 3], [2, 3] |
| S= [1, 2, 3, 4], N = 1 | [1], [2], [3], [4]     |
